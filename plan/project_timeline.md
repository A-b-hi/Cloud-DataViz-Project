## Project Timeline and Milestones

The project is planned and executed over a structured timeline to ensure systematic development, continuous progress, and effective evaluation. The timeline is divided into weekly phases, with each phase focusing on specific deliverables and outcomes.

---

### Week 1: Project Planning and Design
- Finalization of project title and scope  
- Creation of project repository on GitHub  
- Writing abstract, problem statement, and objectives  
- Literature review on data visualization, cloud computing, and virtualization  
- Requirement analysis (functional and non-functional)  
- Technology stack selection  
- System architecture design  
- Project timeline and risk analysis documentation  

**Deliverables:**
- Complete project documentation structure  
- Multiple valid GitHub commits  
- Clean and organized repository  

---

### Week 2: Dataset Creation and Preprocessing
- Selection of dataset topic (business, sales, or academic data)  
- Dataset generation using reliable sources or AI tools  
- Data cleaning and preprocessing  
- Handling missing values and data inconsistencies  
- Dataset documentation  

**Deliverables:**
- Raw dataset  
- Cleaned dataset (CSV format)  
- Dataset description document  

---

### Week 3: Basic Data Visualization
- Importing dataset into visualization tool  
- Creation of basic charts (bar chart, line chart, pie chart)  
- Understanding data trends and patterns  
- Documentation of visual insights  

**Deliverables:**
- Initial visualization files  
- Chart explanation documentation  

---

### Week 4: Interactive Dashboard Development
- Designing a complete interactive dashboard  
- Adding filters and user interaction features  
- Dashboard layout optimization  
- Publishing dashboard locally  

**Deliverables:**
- Interactive dashboard file  
- Dashboard design documentation  

---

### Week 5: Virtualization Setup
- Installation and configuration of virtual machine  
- Deployment of visualization environment inside VM  
- Testing dashboard inside virtual environment  

**Deliverables:**
- Virtual machine setup documentation  
- VM screenshots and configuration files  

---

### Week 6: Cloud Deployment
- Publishing dashboard to cloud platform  
- Enabling remote access through cloud services  
- Testing system accessibility from multiple devices  

**Deliverables:**
- Cloud deployment documentation  
- Live dashboard access link  

---

### Week 7: Documentation and Analysis
- Writing methodology and system design sections  
- Results and analysis documentation  
- Architecture diagram refinement  

**Deliverables:**
- Final report draft  
- System architecture diagram  

---

### Week 8: Finalization and Viva Preparation
- Final system testing  
- Report formatting and submission  
- Preparation of viva questions and answers  
- Final GitHub cleanup and verification  

**Deliverables:**
- Final project report  
- Viva preparation material  
- Complete GitHub repository  

---

This structured timeline ensures systematic project development, continuous assessment, and timely completion.
